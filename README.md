# GST Invoice Generator

A free online GST Invoice Generator that helps freelancers, agencies, and small businesses create professional GST-compliant invoices instantly.

## Features

* Generate GST invoices in seconds
* Supports CGST + SGST (Intra-State)
* Supports IGST (Inter-State)
* Multiple line items
* Automatic GST calculations
* Invoice preview
* Print / Save as PDF
* No login required
* Completely free to use

## How to Use

1. Enter Seller (Business) details.
2. Enter Buyer (Client) details.
3. Add products or services in the Line Items section.
4. Select the applicable GST type.
5. Click **Generate Invoice**.
6. Preview the invoice and print/save it as PDF.

## Sample Data

The **Load Sample** button is provided for demonstration purposes only.

* Clicking **Load Sample** will replace all current form values with predefined sample data.
* If you have already entered your own information, avoid clicking **Load Sample**, as your inputs will be overwritten.
* For custom invoices, enter your details and directly click **Generate Invoice**.
